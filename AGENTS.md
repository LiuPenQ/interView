你是一个富有顶级审美的UX\UI设计师与高级前端工程师双料人才，帮我生成一个个人网站主页设计的trae提示词。

# 主色分析
#295eac 是一种沉稳的深蓝色，带有科技感和专业气质，适合作为品牌主色。
# 完整配色方案
## 1. 核心色彩系统
| 角色             | 色值        | 用途              |
| :------------- | :-------- | :-------------- |
| **主色 Primary** | `#295eac` | 按钮、链接、重点强调、品牌标识 |
| **深主色 Dark**   | `#1a3d7a` | 悬停状态、深色背景、页脚    |
| **浅主色 Light**  | `#4a7fd4` | 悬停高亮、渐变终点、次要按钮  |

## 2. 中性色系统
| 角色       | 色值        | 用途          |
| :------- | :-------- | :---------- |
| **标题文字** | `#0f172a` | 大标题、重要文字    |
| **正文文字** | `#334155` | 段落内容、描述文字   |
| **次要文字** | `#64748b` | 辅助说明、时间戳、标签 |
| **浅色边框** | `#e2e8f0` | 卡片边框、分割线    |
| **背景灰**  | `#f8fafc` | 区块背景、卡片底色   |
| **纯白**   | `#ffffff` | 主背景、卡片表面    |

## 3. 功能色彩
| 角色             | 色值        | 用途        |
| :------------- | :-------- | :-------- |
| **成功 Success** | `#10b981` | 成功提示、完成状态 |
| **警告 Warning** | `#f59e0b` | 警告提示、待处理  |
| **错误 Error**   | `#ef4444` | 错误提示、删除操作 |
| **信息 Info**    | `#0ea5e9` | 信息提示、标签   |
